# GSoC 2025 — OG-CLEWS Preparation

Projects for the **OG–CLEWS** integration proposal
under the United Nations Department of Economic and Social Affairs (UN DESA).

Organization: UNITED NATIONS OFFICE OF INFORMATION AND COMMUNICATIONS TECHNOLOGY
Contributor role: Contributor 1 — Scientific Programming & Model Integration

## Projects

| # | Project | Status | Description |
|---|---|---|---|
| 01 | [OG-Core Runner](./og_runner/) | Complete | Standalone runner with structured output, log capture, and summary extraction |
| 02 | [ETL Pipeline](./etl_pipeline/) | In progress | CLEWS → OG-Core data exchange pipeline |
| 03 | [Validation Framework](./validation_framework/) | Planned | Schema validation for pipeline inputs/outputs |
| 04 | [Convergence Prototype](./convergence_prototype/) | Planned | Iterative coupled model convergence logic |

## Context

These projects demonstrate readiness for the GSoC OG–CLEWS project,
which integrates the CLEWS resource systems model with the OG-Core
macroeconomic model into a unified policy tool for developing countries.

- MUIOGO repo: https://github.com/EAPD-DRB/MUIOGO
- OG-Core repo: https://github.com/PSLmodels/OG-Core
